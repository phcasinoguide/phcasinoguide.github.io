Tongly 隱私權政策

1. 我們收集哪些資料？

Tongly 擴充功能不收集、不儲存、不傳輸任何個人身分資訊。

擴充功能僅在您的裝置本地處理下列資料：

  • 您在支援平台上輸入或接收的聊天文字（僅用於即時翻譯，不儲存）
  • 您在 popup 中設定的語言偏好及 NVIDIA API Key（僅存於 chrome.storage.sync，不傳送至我們的伺服器）

2. 資料如何使用？

聊天文字僅用於即時翻譯，翻譯請求會發送至以下第三方服務：

  • Google Translate API（免費版，無需帳號）— 作為預設翻譯引擎
  • NVIDIA Cloud API（需使用者自行提供 API Key）— 作為高品質翻譯選項

翻譯完成後，原始文字與翻譯結果均不會被儲存於任何伺服器。

3. 我們不做的事

  • 不收集使用者帳號、姓名、Email 或任何個人資訊
  • 不追蹤您的瀏覽行為
  • 不向第三方出售或共享任何資料
  • 不在我們自己的伺服器上儲存任何聊天內容

4. 權限說明

擴充功能需要以下瀏覽器權限：

  • storage：儲存您的語言設定與 API Key 至 Chrome 同步儲存
  • alarms：維持 Service Worker 持續運作（防止翻譯中斷）
  • host_permissions：存取特定平台頁面（WhatsApp、Telegram、蝦皮等）以偵測並翻譯聊天訊息

5. API Key 安全性

若您選擇輸入 NVIDIA API Key，該 Key 僅儲存於您自己的 Chrome 瀏覽器同步儲存中，不會傳送至 Tongly 的任何伺服器。翻譯請求直接從您的瀏覽器發送至 NVIDIA API。

6. 兒童隱私

本擴充功能不針對 13 歲以下兒童，且不刻意收集兒童個人資訊。

7. 政策變更

若本政策有重大變更，我們將更新本頁面的日期並在 Chrome Web Store 更新說明中告知。

8. 聯絡我們

如有任何隱私相關問題，請聯絡：getcrosslingo@gmail.com

Tongly – Privacy Policy

1. What data do we collect?

Tongly does not collect, store, or transmit any personally identifiable information.

The extension only processes the following data locally on your device:

  • Chat text from supported platforms (used only for real-time translation, never stored)
  • Language preferences and NVIDIA API Key set in the popup (stored only in chrome.storage.sync, never sent to our servers)

2. How is data used?

Chat text is used solely for real-time translation. Translation requests are sent to the following third-party services:

  • Google Translate API (free tier, no account required) — default translation engine
  • NVIDIA Cloud API (requires user-provided API Key) — high-quality translation option

After translation, neither the original text nor the translated result is stored on any server.

3. What we do NOT do

  • We do not collect usernames, names, emails, or any personal information
  • We do not track your browsing behavior
  • We do not sell or share any data with third parties
  • We do not store any chat content on our own servers

4. Permissions explained

  • storage: Save your language settings and API Key to Chrome sync storage
  • alarms: Keep the Service Worker alive to prevent translation interruptions
  • host_permissions: Access specific platform pages (WhatsApp, Telegram, Shopee, etc.) to detect and translate chat messages

5. API Key security

If you choose to enter an NVIDIA API Key, it is stored only in your own Chrome browser sync storage and is never sent to Tongly's servers. Translation requests are sent directly from your browser to the NVIDIA API.

6. Children's privacy

This extension is not directed at children under 13 and does not knowingly collect personal information from children.

7. Policy changes

If we make significant changes to this policy, we will update the date on this page and note it in the Chrome Web Store update description.

8. Contact us

For any privacy-related questions, please contact: getcrosslingo@gmail.com
